Calculando de Factorial

Se usa un tipo de función llamada "Recursiva". Es aquella función que se llama asi misma.
function factorial(numero)
{
if(numero>1)
{
return numero * factorial(numero-1);
}
else
return numero;
}

Funciones Predefinidas

Lista de funciones más importantes predefinidas en el lenguaje JS.

  • parseFloat(cadena) = Devuelve un número en coma flotante(con decimales)
  • parseInt(cadena,base) = Devuelve un numero entero según la base que le pasemos como parámetro
  • toString(argumento) = Devuelve cadenas de texto el argumento
  • typeOf(argumento) = Nos devuelve el tipo de dato que le pasemos como parámetro
  • escape(cadena) = Nos devuelve los codigos de escape de cada uno de los caracteres de la cadena
  • unescape(cadena) = Nos devuelve la cadena formada por los caracteres cuyos digitos especificamos
  • isNaN(argumento) = Booleano que nos indica si el argumento es un valor numérico
  • eval(expresion) = Devuelve el resultado de la expresión que le pasemos como parámetro
