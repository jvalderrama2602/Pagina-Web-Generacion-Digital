Butterfly Template

HTML Structure

This HTML Template based on Twitter Bootstrap v3.2.0. It is compatible with a IE9 (with fallback to IE8) browser and all other modern browsers.

It has several main section classes that may or may not present on page (depenging on certain page). You can use it depending on your needs. They are:

  • "light_section" - section with white background.
  • "grey_section" - section with light grey background.
  • "darkgrey_section" section with dark grey background.
  • "dark_section" section section with dark color background.
  • "color_section" section with main template color background.

Also there are some of predefined unique sections (IDs) that you can use with classes above:

  • "header" - contains site logo and site Main Menu.
  • "mainslider" - contains a homepage main slider.
  • "map" - contains a google map.
  • "breadcrumbs" section - contains a site navigation (breadcrumbs).
  • "content" section - used for main content section.
  • "pagination" section - contains a site pagination.
  • "footer" section has a footer widgets.
  • "copyright" section contains a copyright information.
  • "preloader" div element is a page preloader. Remove it if you do not need a preloader on your pages.

CSS Files and Structure

There are four main CSS files in this HTML Template. The first one is a standard Bootstrap v3 CSS file. Second css file is "animation.css" library, which contains various animation classes that are used in this template.

The third file is "main.css" and it contains all of the specific stylings for the pages. The file is separated into following sections:

  • bootstrap overrides - Here we overriding some of predefined bootstrap styles.
  • vertical margins and paddings - Here we set up a vertical margins and paddings for main sections of our HTML Temlpate.
  • general styles - In this section we set up a base template styles, such as links color, general section styles.
  • utilites - some helper classes for margins and paddings etc.
  • styled lists - styles for predefined template lists
  • icons - styles for icons inside buttons, headings and paragraphs
  • buttons - styles for template buttons.
  • common sections styles - common styles for general template sections:
    • light section
    • grey section
    • darkgrey section
    • dark section
    • color section
  • boxed layout - styles for boxed version of template.
  • template styles - styles for this certain template.
  • main menu (min 768px) styles - styles for desctop version of main menu.
  • main menu mobile styles - styles for mobile version of main menu.
  • blog - This section contains styles for a blog page and a single post page.
  • widgets - In this sections we styled all major widgets.
  • shortcodes - In this sections we styled all of template shortcodes.
  • plugins - Styles for plugins that we used in our template:
    • Owl Carousel
    • BxSlider
    • Fraction Slider
    • FlexSlider
    • prettyPhoto
    • Easy pieChart
    • toTop
    • Isotope Portfolio

Fourth file is "fonts.css" that has styles for all of our icon fonts that we used in our template.

Changing wide and boxed layout

To change template layout from wide to boxed you need to do two things:

  1. add "boxed" class to "body" HTML element
  2. add "container" class to div with class "box_wrap"

Example:

<body class="boxed"> 
    <div class="box_wrap container"> 
        ... 
    </div>
</body>

To add pattern in boxed layout just add one of following predefined CSS classes to "body" HTML element:

  1. pattern1
  2. pattern2
  3. pattern3
  4. pattern4

Example:

<body class="boxed pattern2"> 
        ... 
</body>

Changing main color

If you want to change a main color for this template, just replase all instances of main color hex code with a hex color value that you want in the "main.css" file.

Also you can change "main.css" to "main2.css" or "main3.css" which are the same except predefined colors within it.

Changing Ligth and Dark versions

You can change a class for any section inside "#box_wrap" div to change your template version from light to lightgrey or dark.

Available section classes are:

  • light_section
  • grey_section
  • darkgrey_section
  • dark_section
  • color_section

Examples:

<section class="light_section" id="content"> 
        ... 
</section>

You can change to:

<section class="darkgrey_section" id="content"> 
        ... 
</section>

Setup Widgets and Shortcodes.

Wigets and Shortcodes that needs to setup are:

  • Google Map
  • Flickr
  • Twitter
  • Subscribe
  • Contact form
To setup a Google Map do follofing:
  • Scroll to the bottom of HTML page with Google Map (index page and contacts page) and find code " //type your address after "address=" "
  • Provide your address as a regular string.
  • To change a marker text and title find code "//type your map title and description here" and type your values.
To setup a Flickr Widget do follofing:
  • Go to http://idgettr.com/ and search for your Flickr ID
  • Edit line 154 of your "main.js" file. Just replase "54297118@N03" ID with your Flickr ID
To setup a Twitter Widget do follofing:
  • Go to https://dev.twitter.com/apps and register a new application to recieve Consumer Key, Consumer Secret Word, User Token and User Secred Word
  • Edit lines from 27 to 36 of your "twitter/index.php" file. Place your secret keys and words to appropriate placeholders for them.
  • Edit lines 86 (number of tweets) and 91 (username) of your "main.js" file.
To setup a Contact Form do follofing:
  • Edit lines from 7 to 22 of your "contact-form.php" file. Place your email and messages text.
To setup a MailChimp Subscribe form do follofing:

Edit the "mailchimp/store-address.php" file:

Line 19: replace "your_apikey" with an API Key retrieved from here:

http://admin.mailchimp.com/account/api/

Line 23: replace "my_list_unique_id" with the List Unique Id obtained by going to: http://admin.mailchimp.com/lists/.

Click the "settings" link for the list - the Unique Id is at the bottom of that page.

Setup Animation for any HTML elements

To set up animation for single element, just add class "to_animate", attributes "data-animation" with this class, and "data-delay" with number in milliseconds to any HTML element and you're done.

Available values for "data-animation" attribute

  • fadeInLeft
  • fadeInUp
  • fadeInRight
  • slideRight
  • slideUp
  • slideLeft
  • expandUp
  • slideExpandUp
  • fadeIn
  • expandOpen
  • bigEntrance

  • pulse
